# 学习调整

当用户给出调整建议时，记录到文档中。

## 触发场景

用户说：
- "以后遇到 X 场景，记得用 Y 方式"
- "这个调整一下"
- 给出任何改进建议

## 操作步骤

### 1. 理解建议

- 确认理解用户的调整内容
- 理解背后的原因

### 2. 记录到文档

保存到 `.claude/learnings/adjustments.md`：

```markdown
# 学习记录

## 调整 [日期]

### 内容
[用户的调整建议]

### 原因
[理解后的原因]

### 应用场景
[什么时候应该应用]
```

### 3. 应用

当遇到类似场景时：
1. 读取 `.claude/learnings/adjustments.md`
2. 应用调整
3. 如果不确定，可以问用户

## 记录原则

- 简洁明了
- 突出应用场景
- 便于后续查找
