# 用户画像查看工具

> 查看和管理 AI 学习到的用户偏好数据

## 使用方式

```
Skill: ai-assistant:user-profile
```

## 功能

### 1. 查看用户画像

显示收集到的用户偏好：
- 代码风格偏好
- 验证要求
- 沟通偏好

### 2. 查看任务统计

显示任务执行统计：
- 总任务数
- 完成任务数
- 失败任务数
- 各类型任务分布

### 3. 查看行为模式

显示学习到的行为模式：
- 常用的执行流程
- 干预习惯
- 成功/失败模式

### 4. 重置数据

清除所有学习数据（谨慎使用）

## 数据位置

```
~/.claude/ai-assistant/
├── user-profile.json    # 用户画像
├── task-history/       # 任务历史
└── patterns/          # 学习到的模式
```
