# 调度模板

> Coordinator 调度过程中使用的模板集合

---

## 模板列表

| 模板 | 文件 | 用途 |
|------|------|------|
| 调度记录 | `调度记录.md` | 任务全生命周期记录 |
| 需求摘要 | `需求摘要.md` | 需求分析产出 |
| 执行计划 | `执行计划.md` | 实施计划产出 |

---

## 使用方式

在编写调度文档时，参考对应模板：

```markdown
> 参考：templates/调度记录.md
```

---

## 模板规范

- 使用 Markdown 格式
- 占位符使用 `{占位符名称}` 格式
- 可选字段用 `(可选)` 标注
