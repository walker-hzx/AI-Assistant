## 检查点列表

## 检查函数

### 通用检查函数

```python
def check_document_exists(path):
    """检查文档是否存在"""
    return Glob(path).exists()

def check_document_fields(path, required_fields):
    """检查文档必填字段"""
    content = Read(path)
    for field in required_fields:
        if field not in content:
            return False, f"缺失字段: {field}"
    return True, "检查通过"
```

### 使用方式

```
【检查点 N】
1. 使用 Glob 工具检查文档是否存在
2. 使用 Read 工具读取文档内容
3. 检查必填字段是否存在
4. 根据检查结果决定下一步
```

---

## 失败处理原则

| 失败类型 | 处理方式 |
|----------|----------|
| 文档缺失 | 创建后重试 |
| 字段缺失 | 补充后重试 |
| 验证失败 | 修复后重试（最多 3 次） |
| 审查失败 | 修复后重试或接受风险 |
| 达到上限 | 通知用户，人工介入 |
