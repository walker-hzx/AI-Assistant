**任务描述**：
根据需求文档创建简单的 HTML 登录页面，包含：
- 用户名输入框（type="text"）
- 密码输入框（type="password"）
- 登录按钮（type="submit"）
- 简单的表单布局结构

**验收标准对照**：

| 登录按钮（type="submit"） | `<button type="submit">登录</button>` 或 `<input type="submit">` |
| 简单表单布局 | `<form>` 标签包裹输入框和按钮 |

**输出文件**：
- `tests/samples/login.html`

**边界条件**：
- 输入框应有 name 属性以便后续表单提交
- 表单应有 action 属性（可为空）
- 保持 HTML 结构简洁清晰
