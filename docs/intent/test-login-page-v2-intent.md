# 需求文档：简单用户登录页面

## 核心问题

**解决什么**：为系统提供用户身份认证的入口界面

**一句话总结**：创建一个包含用户名、密码输入框和登录按钮的简单 HTML 登录页面

---

## 用户场景

| 场景 | 描述 |
|------|------|
| 场景1 | 用户访问系统登录页面，输入用户名和密码进行身份认证 |

**目标用户**：需要登录系统的普通用户

**使用场景**：
- 用户首次访问系统
- 用户重新登录
- Session 过期后重新登录

---

## 功能范围

### 要做 (Do)

| 功能 | 描述 |
|------|------|
| 用户名输入框 | type="text" 的文本输入框 |
| 密码输入框 | type="password" 的密码输入框 |
| 登录按钮 | type="submit" 的提交按钮 |
| 表单布局 | 简单的表单布局结构 |

### 不做 (Don't)

| 功能 | 原因 |
|------|------|
| 登录逻辑 | 需求明确仅做页面，不含后端逻辑 |
| 表单验证 | 简单的 HTML 页面，无需 JS 验证 |
| 样式美化 | 需求明确"简单表单布局" |
| 后端接口 | 不包含任何后端服务 |
