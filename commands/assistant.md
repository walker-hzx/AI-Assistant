# 智能助手

使用此命令让管家帮你处理需求。

## 【重要】使用规则

**当你被此命令调用时，必须调用对应的 skill：**

```
使用 Skill 工具调用：
- skill: ai-assistant:assistant
```

**你不能直接回答用户，必须先调用 assistant skill。**

---

## 使用方式

```
/assistant [你的需求]

例如：
/assistant 帮我加个用户登录功能
/assistant 修复登录页面的拼写错误
/assistant 帮我优化下接口性能
```

## 功能

1. 接收需求，判断复杂度
2. 派发给合适的角色处理
3. 监控进度，收集结果
4. 输出状态提示，生成文档

## 典型场景

| 场景 | 输入 | 管家会 |
|------|------|--------|
| 新功能 | `/assistant 加个用户管理功能` | 需求分析 → 计划 → 执行 → 验证 |
| Bug修复 | `/assistant 登录接口报错了` | 调试 → 修复 → 验证 |
| 代码优化 | `/assistant 优化下用户查询性能` | 分析 → 优化 → 验证 |
| 简单任务 | `/assistant 改个变量名` | 直接处理，记录结果 |
