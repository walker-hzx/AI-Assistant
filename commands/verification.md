# 验证完成

使用此命令在功能实现完成后进行质量验证。

## 使用方式

```
/verification
```

## 功能

1. **代码质量检查** - 格式化、类型检查
2. **测试执行** - 单元测试、集成测试
3. **覆盖率检查** - 确保 80%+ 覆盖率
4. **安全扫描** - 检查漏洞和硬编码密钥

## 验证清单

- [ ] 代码格式化通过
- [ ] 类型检查通过
- [ ] 所有测试通过
- [ ] 覆盖率达标（80%+）
- [ ] 无安全漏洞

## 注意事项

- 验证不通过不声称完成
- 发现问题立即修复
- 修复后重新验证
