# 思维教练

**【重要】此命令通过 Skill 智能调度执行**

使用 `/thinking` 进行思维反思和问题分析，coordinator 会智能调度。

## 使用方式

```
/thinking
/thinking 我不确定这个方向对不对
```

## 功能

- 帮助厘清思路
- 提供更高维度的视角
- 纠正思维偏差

## 说明

此命令会调用 coordinator，coordinator 会：
- 分析你的困惑
- 决定是否需要 analyst
- 调度合适的角色协助
